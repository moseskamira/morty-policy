Privacy Policy

Last updated: May 2026

MortyVerse is a fan-based mobile application that displays information about characters, episodes, and locations from the Rick and Morty universe using a public API.

Information Collection

This app does not collect, store, or share any personal information.

No user accounts, registrations, or sign-ins are required to use the app.

The app does not collect:

  • Names
  • Email addresses
  • Phone numbers
  • Location data
  • Any personal identifiers

Third-Party Services

The app retrieves data from a public API to display characters, episodes, and related content. These services may operate independently under their own policies.

Data Usage

Since no personal data is collected, nothing is stored or shared by the app.

Security

The app does not transmit or store sensitive user data. All network requests are only used to fetch publicly available API information.

Children’s Privacy

This app does not knowingly collect any data from children or any users.
